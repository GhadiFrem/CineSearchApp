CineSearchReact
CineSearchReact is a sleek, high-performance movie discovery application built with React.js and Vite. It provides a seamless interface for users to browse the latest blockbusters, search for specific titles, and dive deep into movie metadata including plots, genres, and release years.

 Features
Real-time Search: Instantly find movies by title.
Dynamic Details: View rich data including posters, plot summaries, and ratings.
Fast Builds: Powered by Vite for an ultra-fast development experience.
Clean UI: Minimalist aesthetic focused on movie posters and readability.
API: OMDB API

 Tech Stack
Frontend: React.js
Build Tool: Vite
Styling: CSS3
